## Installation

1. Clone the repository
```bash
git clone https://github.com/ambonxnobody/LaravelJetstream-RumahDrone.git
```

2. Install the dependencies
```bash
composer install
```

3. Setup a new database

4. Copy the `.env.example` file to `.env` and update the database configuration

5. Run the migrations
```bash
php artisan migrate:fresh --seed
```

6. Install the NPM dependencies
```bash
npm install
```

7. Build the assets
```bash
npm run dev
```

8. Serve the application
```bash
php artisan serve
```

9. Visit the application in your browser at `http://localhost:8000`
