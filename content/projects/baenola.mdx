# Baenola

A simple, clean, and responsive CMS website for Baenola.

## Features

- **Responsive**: The website is fully responsive and works on all devices.
- **CMS**: The website is built with a CMS, so the client can easily update the content.
- **SEO**: The website is optimized for search engines.
- **Performance**: The website is optimized for performance.

## Technologies

- **React**: The website is built with React framework.
- **Laravel**: The CMS is built with Laravel framework.
